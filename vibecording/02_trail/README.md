# 🖱️ Image Trail — 마우스 트레일 이펙트

## 📌 개요
마우스를 움직이면 컬러풀한 카드 이미지가 뱀처럼 줄지어 따라다니는 시각 효과입니다. 클릭하면 Trail 모드와 Grid 모드를 전환할 수 있습니다.

## 🛠 사용 기술
| 기술 | 용도 |
|---|---|
| **HTML5 Canvas** | 2D 이미지 렌더링 |
| **JavaScript (Vanilla)** | 마우스 추적 + 애니메이션 |
| **Canvas toDataURL()** | 이미지를 코드로 동적 생성 |

> 외부 이미지 파일 없이, 코드에서 그라데이션 카드를 자동 생성합니다.

## 🎯 핵심 기법

### 1. 마우스 추적 + 스폰 거리
```
마우스가 일정 거리(80px) 이상 이동했을 때만 새 카드를 생성
→ 너무 많은 카드가 한꺼번에 나오는 것을 방지
```

### 2. 뱀 꼬리 따라가기 (Follow-chain)
```
첫 번째 카드 → 마우스를 따라감
두 번째 카드 → 첫 번째 카드를 따라감
세 번째 카드 → 두 번째 카드를 따라감
... (체인 구조)
```
각 카드는 `ease` 값이 점점 작아져서 뒤쪽 카드일수록 느리게 따라갑니다.

### 3. 페이드인/아웃
- 새 카드: scale 0 → 1로 팝업 + 흰색 플래시
- 오래된 카드: alpha와 scale이 서서히 줄어들며 사라짐

### 4. 동적 이미지 생성
```javascript
// Canvas로 그라데이션 이미지를 만들어서 사용
const offCanvas = document.createElement('canvas');
const grad = offCtx.createLinearGradient(...);
img.src = offCanvas.toDataURL();  // 이미지 URL로 변환
```

### 5. 터치 지원
`touchmove`, `touchstart` 이벤트로 모바일에서도 동작합니다.

## 💬 이런 프롬프트로 만들어보세요

### 기본 버전
```
검은 배경에서 마우스를 움직이면 작은 컬러 사각형이
마우스를 따라다니게 만들어줘.
HTML Canvas로 만들어줘.
```

### 업그레이드 1: 트레일 체인
```
사각형이 하나가 아니라 여러 개가 뱀처럼 줄지어 따라오게 해줘.
앞의 것을 뒤의 것이 따라가는 체인 구조로 만들어줘.
최대 14개까지만 보이게 하고, 오래된 건 사라지게 해줘.
```

### 업그레이드 2: 카드 스타일
```
사각형 대신 그라데이션 카드 이미지로 바꿔줘.
각 카드에 그림자와 살짝 기울어진 회전을 넣어줘.
새로 나올 때 흰색 플래시 효과를 추가해줘.
```

### 업그레이드 3: 그리드 모드 추가
```
클릭하면 따라다니던 카드들이 원형으로 정렬되는
그리드 모드로 전환되게 해줘.
다시 클릭하면 트레일 모드로 돌아가게 해줘.
```

## 📚 학습 포인트
- 마우스 이벤트 (`mousemove`) 처리
- 체인 기반 추적 알고리즘 (Follow-the-leader)

- `toDataURL()`을 이용한 동적 이미지 생성
- 상태 전환 (Trail ↔ Grid) 패턴
