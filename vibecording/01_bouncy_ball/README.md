# 🏀 Bouncy Ball — 물리 바운스 이펙트

## 📌 개요
원형 컨테이너 안에서 공이 튀어다니며, 7,800개의 프리즘 파티클이 공을 자석처럼 밀어내는 인터랙티브 물리 시뮬레이션입니다.

## 🛠 사용 기술
| 기술 | 용도 |
|---|---|
| **HTML5 Canvas** | 2D 그래픽 렌더링 |
| **JavaScript (Vanilla)** | 물리 엔진 + 애니메이션 |
| **Web Audio API** | 충돌 시 사운드 효과 |

> 외부 라이브러리 없이 순수 HTML + JS로만 구현되었습니다.

## 🎯 핵심 기법

### 1. 물리 시뮬레이션
```
중력(gravity) → 공이 아래로 떨어짐
반발 계수(bounce) → 벽에 부딪히면 튕겨나감
공기저항(friction) → 속도가 자연스럽게 감소
```

### 2. 벡터 반사 (충돌 처리)
공이 원형 벽에 부딪히면 **법선 벡터 기준으로 속도를 반사**합니다.
```
반사 공식: V' = V - 2(V·N)N
```

### 3. 스쿼시 & 스트레치
- 충돌 순간: 공이 **찌그러짐** (squash)
- 이동 중: 공이 **늘어남** (stretch)
- 이 효과가 공에 생동감을 줍니다

### 4. 파티클 상호작용
- 7,800개 파티클이 공과 **일정 거리 이내**에 있으면 밀려남
- 가까울수록 **흰색으로 변화** (프리즘 효과)

### 5. 사운드 피드백
- Web Audio API로 충돌 시 **실시간 사운드 생성**
- 컨테이너 크기에 따라 음높이가 변함 (작으면 고음, 크면 저음)

## 💬 이런 프롬프트로 만들어보세요

### 기본 버전
```
검은 배경에 흰색 공을 하나 만들어줘.
중력으로 아래로 떨어지고, 바닥에 부딪히면 튕겨 올라가게 해줘.
HTML Canvas로 만들어줘.
```

### 업그레이드 1: 원형 컨테이너
```
바닥 대신 원형 컨테이너 안에서 튕기게 바꿔줘.
원의 벽에 부딪히면 반사되도록 해줘.
```

### 업그레이드 2: 파티클 추가
```
원 안에 작은 컬러 파티클 1000개를 추가해줘.
파티클은 공이 가까이 오면 밀려나게 해줘.
HSL 컬러로 무지개 색을 입혀줘.
```

### 업그레이드 3: 사운드
```
공이 벽에 부딪힐 때 Web Audio API로 톤 소리를 내줘.
충돌 강도에 따라 음높이를 다르게 해줘.
```

## 📚 학습 포인트
- Canvas 2D `arc()`, `fillRect()` 기본 도형 그리기
- `requestAnimationFrame()` 기반 게임 루프
- 2D 벡터 수학 (거리, 내적, 반사)
- Web Audio API (`OscillatorNode`, `GainNode`)
- 물리 시뮬레이션의 기초 (중력, 반발, 마찰)
