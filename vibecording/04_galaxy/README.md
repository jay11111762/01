# 🌌 Galaxy Scroll — 은하수 타이포그래피

## 📌 개요
수천 개의 문자 파티클이 우주 먼지처럼 떠다니다가, 스크롤하면 글자 형태로 모여들고, 더 스크롤하면 다시 흩어지는 3단계 인터랙티브 효과입니다.

## 🛠 사용 기술
| 기술 | 용도 |
|---|---|
| **HTML5 Canvas** | 2D 파티클 렌더링 |
| **JavaScript (Vanilla)** | 파티클 시스템 + 스크롤 제어 |
| **Google Fonts (Inter)** | 웹폰트 로딩 |
| **캔버스 비트맵 분석** | 텍스트 → 파티클 위치 변환 |

## 🎯 핵심 기법

### 1. 텍스트 → 파티클 좌표 변환
```
1) 보이지 않는 캔버스에 텍스트를 그림
2) 픽셀 데이터를 읽어 불투명한 곳의 좌표를 추출
3) 그 좌표들이 파티클의 "목표 위치"가 됨
```
이 기법으로 어떤 텍스트든 파티클 형태로 표현할 수 있습니다.

### 2. 3단계 스크롤 전환
```
Phase 1 (0~33%): Chaos — 파티클이 무작위로 떠다님
Phase 2 (33~66%): Order — 파티클이 글자 형태로 모임
Phase 3 (66~100%): Dissolution — 다시 흩어짐
```

### 3. Lerp (선형 보간) 애니메이션
```javascript
// 현재 좌표에서 목표 좌표까지 부드럽게 이동
x += (targetX - x) * ease;
```
이 한 줄이 부드러운 모션의 핵심입니다.

### 4. 깊이감 (Z-axis 시뮬레이션)
각 파티클에 `z` 값을 부여하여:
- 가까운 파티클: 크고 밝게
- 먼 파티클: 작고 어둡게
→ 2D Canvas에서 3D 느낌을 만듭니다.

### 5. 마우스 휠 + 터치 스크롤 지원
물리적 스크롤이 아닌 **가상 스크롤 값**을 별도로 관리합니다.

## 💬 이런 프롬프트로 만들어보세요

### 기본 버전
```
검은 배경에 작은 흰색 점 500개를 랜덤으로 뿌려줘.
각 점이 천천히 떠다니게 해줘.
HTML Canvas로 만들어줘.
```

### 업그레이드 1: 텍스트 형태로 모이기
```
Canvas에 "HELLO"라는 텍스트를 보이지 않게 그리고,
텍스트의 픽셀 위치를 추출해서 파티클의 목표 위치로 설정해줘.
스크롤하면 파티클이 글자 형태로 모이게 해줘.
```

### 업그레이드 2: 3단계 전환
```
스크롤 진행도에 따라 3단계로 만들어줘:
- 0~33%: 파티클이 자유롭게 떠다님
- 33~66%: 글자 형태로 모임
- 66~100%: 다시 흩어짐
각 전환이 부드럽게 lerp로 이루어지게 해줘.
```

### 업그레이드 3: 깊이감
```
각 파티클에 z값을 줘서 3D 느낌을 내줘.
가까운 건 크고 밝게, 먼 건 작고 어둡게 표현해줘.
빨간색으로 발광하는 파티클도 일부 섞어줘.
```
