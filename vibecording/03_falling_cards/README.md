# 🃏 Falling Cards — 카드 낙하 시계

## 📌 개요
현재 시간(HH:MM:SS)을 6개의 카드 스택으로 표현합니다. 매 초마다 숫자가 바뀌면 기존 카드가 중력을 받아 아래로 떨어지고, 새 카드가 쌓입니다. 실로폰 같은 사운드도 함께 재생됩니다.

## 🛠 사용 기술
| 기술 | 용도 |
|---|---|
| **HTML + CSS** | 카드 UI + 떨어지는 애니메이션 |
| **CSS Transitions** | 부드러운 낙하 모션 |
| **JavaScript (Vanilla)** | 시간 계산 + 카드 관리 |
| **Google Fonts** | Bebas Neue 폰트 |
| **Web Audio API** | 실로폰 사운드 효과 |

> Canvas 없이 **DOM + CSS만으로** 모션을 구현한 예제입니다!

## 🎯 핵심 기법

### 1. CSS로 물리 모션 흉내내기
```css
/* 중력을 받아 떨어지는 느낌의 CSS transition */
transition: transform 1.2s cubic-bezier(0.25, 0.46, 0.45, 0.94);
```
JavaScript의 물리 계산 없이 **cubic-bezier 커브**만으로 중력 낙하를 표현합니다.

### 2. 실시간 시계 로직
```
매 초마다 현재 시간을 읽음
→ 이전 숫자와 비교
→ 변한 자리만 카드를 교체
```

### 3. 카드 스택 관리
- 각 자리(HH:MM:SS)에 최대 8장의 카드가 쌓임
- 새 카드는 위에 추가, 기존 카드는 아래로 밀림
- 오래된 카드는 DOM에서 제거 (메모리 관리)

### 4. 지그재그 배치
카드가 쌓일 때 약간의 **각도와 X 오프셋**을 번갈아 적용하여 자연스러운 스택 느낌을 줍니다.

### 5. 컬러 팔레트 업데이트
현재 보이는 최상단 카드의 색상이 주기적으로 바뀌어 시각적 리듬을 만듭니다.
