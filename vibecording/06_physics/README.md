# ⚡ Physics Typography — 물리 타이포그래피

## 📌 개요
화면에 "PHYSICS" 같은 텍스트가 크게 표시되어 있고, 마우스를 가져가면 글자들이 물리 법칙에 따라 **밀려나고 흩어집니다**. 시간이 지나면 다시 원래 위치로 돌아옵니다.

## 🛠 사용 기술
| 기술 | 용도 |
|---|---|
| **HTML5 Canvas** | 2D 글자 렌더링 |
| **JavaScript (Vanilla)** | 물리 엔진 + 마우스 인터랙션 |
| **Google Fonts (Inter)** | 웹폰트 |
| **캔버스 비트맵 분석** | 텍스트 → 파티클 위치 변환 |

## 🎯 핵심 기법

### 1. 텍스트를 물리 오브젝트로 변환
```
1) Canvas에 큰 텍스트를 그림
2) 픽셀 데이터를 분석하여 불투명한 위치 추출
3) 각 위치에 물리 속성을 가진 글자 파티클 배치
```

### 2. 마우스 반발력 (Repulsion)
```javascript
// 마우스와의 거리 계산
const dist = Math.sqrt(dx*dx + dy*dy);
// 가까울수록 더 강한 밀어내는 힘
const force = (repelRadius - dist) / repelRadius;
```
마우스가 가까이 가면 글자들이 **폭발하듯 밀려남**

### 3. 문자 그룹 시스템
같은 단어의 글자들은 **그룹으로 묶여** 있어서:
- 그룹 내 글자끼리 서로 당기는 힘이 있음
- 마우스가 충분히 가까워야 그룹이 "깨짐" (breakGroup)
- 깨진 후 4.5초 뒤 다시 원래 위치로 돌아옴

### 4. 귀환 시스템 (Homing)
```
글자가 흩어진 후 일정 시간이 지나면
→ 원래 위치(homeX, homeY)를 향해 서서히 되돌아감
→ 되돌아가면서 자동으로 그룹도 복구
```

### 5. 캔버스 비트맵 캐싱
각 글자의 비트맵을 미리 렌더링해두고 재사용하여 성능을 확보합니다.

### 기본 버전
```
검은 배경에 "HELLO"를 큰 글씨로 화면 중앙에 표시해줘.
HTML Canvas에 Inter 폰트로 그려줘.
```

### 업그레이드 1: 글자를 파티클로 분해
```
텍스트를 캔버스에 그리고, 픽셀 데이터를 분석해서
글자가 있는 위치마다 작은 문자 파티클을 배치해줘.
각 파티클은 개별적으로 움직일 수 있게 해줘.
```

### 업그레이드 2: 마우스 반발
```
마우스를 글자 위에 올리면 주변 글자 파티클들이
밀려나게 해줘. 가까울수록 더 세게 밀려나게 하고,
마우스에서 멀어지면 다시 원래 위치로 돌아가게 해줘.
```

### 업그레이드 3: 물리 효과 강화
```
밀려난 글자에 관성을 추가해줘.
바닥에 닿으면 중력의 영향을 받게 해줘.
글자간 충돌도 추가해줘.
일정 시간 후 원래 자리로 부드럽게 돌아오게 해줘.
```
