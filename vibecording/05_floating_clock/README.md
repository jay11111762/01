# ⏰ Floating Clock — 3D 플로팅 시계

## 📌 개요
현재 시간을 3D 타이포그래피로 표현합니다. 숫자들이 무중력 상태로 떠다니며 서로 부딪히고, 스캐닝 빔이 지나가면서 글자를 발광시킵니다. 클릭하면 컬러 테마가 바뀝니다.

## 🛠 사용 기술
| 기술 | 용도 |
|---|---|
| **Three.js** | 3D 렌더링 엔진 (CDN 로드) |
| **WebGL Shader** | 커스텀 스캐닝 빔 효과 |
| **3D Font (Typeface)** | Helvetiker Bold 폰트 |
| **JavaScript (Vanilla)** | 물리 시뮬레이션 + 시간 |

> ⚠️ 이 예제는 **Three.js 라이브러리**를 사용합니다 (CDN에서 자동 로드).

## 🎯 핵심 기법

### 1. Three.js 3D 텍스트
```
Three.js FontLoader로 3D 폰트를 로드
→ TextGeometry로 3D 메시 생성
→ 각 숫자가 독립적인 3D 오브젝트
```

### 2. 2D 물리 시뮬레이션 (3D 공간에서)
숫자들이 XY 평면에서 물리 법칙에 따라 이동:
- **관성**: 속도가 서서히 감소
- **벽 충돌**: 화면 가장자리에서 튕김
- **오브젝트 간 충돌**: 서로 부딪히면 밀려남

### 3. 커스텀 GLSL 셰이더
```glsl
// 스캐닝 빔이 지나가면 글자가 밝아지는 효과
float scanEffect = smoothstep(scanWidth, 0.0, dist);
color += rimColor * scanEffect;
```
WebGL 셰이더로 GPU에서 직접 렌더링하여 성능이 뛰어납니다.

### 4. 컬러 테마 전환
6가지 테마(RED, BLUE, GREEN, PURPLE, GOLD, CYAN)가
5초마다 자동으로 또는 클릭 시 부드럽게 전환됩니다.

### 5. 와이어프레임 도형
숫자와 함께 **와이어프레임 다면체**(구, 정이십면체, 정팔면체, 토러스)가 떠다니며 장식 역할을 합니다.

## 💬 이런 프롬프트로 만들어보세요

### 기본 버전
```
Three.js를 사용해서 화면에 현재 시간을 3D 텍스트로 보여줘.
검은 배경에 흰색 글씨로 해줘.
CDN으로 Three.js를 로드해줘.
```

### 업그레이드 1: 플로팅 물리
```
각 숫자가 따로 떨어져서 화면 안에서 떠다니게 해줘.
서로 부딪히면 튕겨나가고, 화면 가장자리에서도 튕기게 해줘.
```

### 업그레이드 2: 스캐닝 빔
```
왼쪽에서 오른쪽으로 빛줄기가 반복해서 지나가게 해줘.
빛줄기가 글자를 지나갈 때 글자가 빨간색으로 빛나게 해줘.
GLSL 셰이더로 구현해줘.
```

### 업그레이드 3: 테마 전환
```
클릭하면 빨강→파랑→초록→보라→금→청록 순서로
컬러 테마가 부드럽게 전환되게 해줘.
5초마다 자동 전환도 추가해줘.
```

## 📚 학습 포인트
